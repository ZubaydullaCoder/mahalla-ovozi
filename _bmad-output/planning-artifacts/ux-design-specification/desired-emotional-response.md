# Desired Emotional Response

## Primary Goal: Calm Authority

The dashboard should feel like a reliable briefing surface, not an emergency
operations center. Stable geometry, restrained accents, predictable
interactions, and clear evidence hierarchy help the user feel oriented.

## Secondary Goal: Informed Trust

Trust comes from visible provenance:

- AI summaries are labeled and visually distinct from resident text.
- Claims remain attributed and uncertain.
- Original evidence is always available in the drawer.
- Exact Telegram links allow source verification when access permits.
- Contradictions remain visible rather than being smoothed into a false fact.

## Emotional Risks

| Risk | Design response |
|---|---|
| Duplicate lane copies look like different incidents | Same summary, categories, mahalla, and canonical drawer identity |
| AI summary looks authoritative | Attribution language and distinct summary/evidence styling |
| Delay looks like data loss | Amber banner plus retained cached topics |
| Evidence becomes overwhelming | Compact cards, chronological drawer, centered anchor |
| Priority looks like severity | Neutral Hokim-lane styling and no severity labels |

## Principles

1. **Authority, not alarm.**
2. **Evidence, not abstraction.**
3. **Attribution, not invented certainty.**
4. **Consistency, not surprise.**
5. **Quiet polish, not showmanship.**
